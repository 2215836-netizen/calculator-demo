# PRD: 공학용 계산기 (Scientific Calculator)

## 1. 프로젝트 개요

### 1.1 목적
모던하고 직관적인 UI/UX를 갖춘 웹 기반 공학용 계산기를 개발하여, 학생, 엔지니어, 과학자들이 기본 연산과 과학 계산을 쉽게 수행할 수 있도록 합니다.

### 1.2 목표
- 사용자 친화적인 인터페이스 제공
- 기본 연산 및 과학 함수 지원
- 반응형 디자인으로 모든 디바이스에서 사용 가능
- 빠르고 정확한 계산 결과 제공

### 1.3 대상 사용자
- 대학생 및 고등학생 (수학, 물리, 화학 과목)
- 엔지니어 및 연구원
- 일반 사용자 (복잡한 계산이 필요한 경우)

## 2. 기능 요구사항

### 2.1 핵심 기능

#### 2.1.1 기본 연산
- **덧셈 (+)**: 두 수의 합 계산
- **뺄셈 (-)**: 두 수의 차 계산
- **곱셈 (×)**: 두 수의 곱 계산
- **나눗셈 (÷)**: 두 수의 나눗셈 계산
- **퍼센트 (%)**: 백분율 계산

#### 2.1.2 과학 함수
- **삼각 함수**
  - `sin`: 사인 함수
  - `cos`: 코사인 함수
  - `tan`: 탄젠트 함수
- **로그 함수**
  - `ln`: 자연로그 (밑이 e)
  - `log`: 상용로그 (밑이 10)
- **거듭제곱 및 루트**
  - `x²`: 제곱 계산
  - `√`: 제곱근 계산
- **역수**
  - `1/x`: 역수 계산
- **수학 상수**
  - `π`: 원주율 (3.14159...)
  - `e`: 자연상수 (2.71828...)

#### 2.1.3 각도 모드
- **Rad (라디안)**: 삼각 함수를 라디안 단위로 계산
- **Deg (도)**: 삼각 함수를 도 단위로 계산
- 모드 전환 버튼으로 쉽게 변경 가능

#### 2.1.4 입력 및 편집
- **숫자 입력**: 0-9 버튼
- **소수점 입력**: `.` 버튼
- **AC (All Clear)**: 모든 입력 및 결과 초기화
- **백스페이스**: 마지막 입력 문자 삭제
- **키보드 지원**: 물리 키보드로도 입력 가능

#### 2.1.5 디스플레이
- **현재 입력/결과**: 큰 글씨로 현재 계산 결과 표시
- **계산 과정**: 작은 글씨로 입력한 수식 표시
- **천 단위 구분**: 큰 숫자는 쉼표로 구분 (예: 12,345.67)

#### 2.1.6 히스토리
- **계산 기록**: 이전 계산 내역 저장
- **히스토리 버튼**: 상단 우측 아이콘으로 접근

### 2.2 부가 기능
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **다크 모드**: 눈의 피로를 줄이는 다크 테마
- **애니메이션**: 버튼 클릭 시 부드러운 피드백 효과

## 3. 비기능 요구사항

### 3.1 성능
- 계산 응답 시간: 100ms 이내
- 페이지 로딩 시간: 2초 이내
- 부드러운 애니메이션 (60fps)

### 3.2 사용성
- 직관적인 버튼 배치
- 명확한 시각적 피드백 (호버, 클릭 효과)
- 접근성: 키보드만으로도 모든 기능 사용 가능

### 3.3 호환성
- **브라우저**: Chrome, Firefox, Safari, Edge (최신 2개 버전)
- **디바이스**: 모바일, 태블릿, 데스크톱
- **화면 크기**: 320px ~ 2560px

### 3.4 신뢰성
- 정확한 계산 결과 (부동소수점 오차 최소화)
- 에러 처리 (0으로 나누기, 잘못된 입력 등)

## 6. 사용자 시나리오

### 6.1 시나리오 1: 기본 계산
**사용자**: 고등학생  
**목적**: 간단한 사칙연산 수행

1. 계산기 웹 페이지 접속
2. `5 + 3 =` 입력
3. 결과 `8` 확인
4. AC 버튼으로 초기화

### 6.2 시나리오 2: 삼각 함수 계산
**사용자**: 공학도  
**목적**: sin(30°) 계산

1. 각도 모드를 `Deg`로 설정
2. `sin` 버튼 클릭
3. `30` 입력
4. `=` 버튼 클릭
5. 결과 `0.5` 확인

### 6.3 시나리오 3: 복잡한 수식
**사용자**: 연구원  
**목적**: (π × 5²) ÷ 2 계산

1. `π` 버튼 클릭
2. `×` 버튼 클릭
3. `5` 입력
4. `x²` 버튼 클릭
5. `÷` 버튼 클릭
6. `2` 입력
7. `=` 버튼 클릭
8. 결과 `39.27` 확인
